Claude Chat Export

Exports the current Claude.ai conversation as a Markdown file with YAML frontmatter.

Install

Drag this button to your bookmarks bar:

Export Chat ← drag to bookmarks bar

Show bookmarks bar

  • Safari: View > Show Bookmarks Bar
  • Brave: View > Show Bookmarks Bar (or Ctrl+Shift+B / Cmd+Shift+B)

Usage

  1. Open a conversation on claude.ai
  2. Click the Export Chat bookmark
  3. Confirm or edit the date (YYYY-MM-DD) and slug when prompted
  4. The .md file downloads automatically

Output filename format: YYYY-MM-DD_Chat_Transcript_Slug.md

Troubleshooting

"Could not find org ID"
Make sure you are logged into claude.ai before running the bookmarklet.
"Session expired"
Refresh the claude.ai page and try again.
"Could not fetch conversation"
The internal API may have changed. Check for an updated version of this bookmarklet.
Nothing happens (Safari)
Go to Settings > Websites > JavaScript and make sure JavaScript is allowed for claude.ai.
Nothing happens (Brave)
Check Brave Shields settings for claude.ai. Try lowering shields or allowing scripts.
