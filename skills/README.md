# Skills Layout

This folder currently keeps both Claude-oriented originals and Codex-oriented ports.

- Unsuffixed folders like `start-session/` and `wrap-up/` are the original Claude-facing skills.
- `*-codex/` folders are Codex-native versions of the same command concept.

The goal is to keep Claude's existing breadcrumbs intact while giving Codex versions their own implementation path.
